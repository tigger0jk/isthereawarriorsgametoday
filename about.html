This site created by Patrick Fay. Originally created to know Is there a Warriors game tonight?. Extended to support other NBA teams.

This site inspired by (and forked from) isthereagiantsgametoday.com

Have an idea? Have I done your team wrong? Colors bad? Font could be better? Fork this project on Github and submit pull requests, report issues, or make suggestions.

The times listed on this site shows the game time in your team's home time zone, even for away games.
